# Currency Converter

Автор: Айзат Шайхутдинов

## Описание
Программа для конвертации валют с графическим интерфейсом, использованием внешнего API и сохранением истории операций.

## Требования
- Python 3.8+
- Библиотеки: requests, tkinter

## Установка
1. Установите зависимости:
   pip install -r requirements.txt

2. Получите API-ключ на exchangerate-api.com и вставьте его в main.py.

## Запуск
python main.py

## Пример использования
1. Выберите валюты.
2. Введите сумму.
3. Нажмите "Конвертировать".
4. История появится в таблице.
